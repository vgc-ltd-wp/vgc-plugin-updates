# VGC Plugin Updates

Public update channel for VGC WordPress plugins. Holds:

- `plugins.json` — manifest mapping each plugin slug to its latest version + a download URL.
- Release assets — the WordPress-ready zip for each published version.

The plugins' **source code** lives in separate (private) repos. This repo only
distributes the built zips so sites can show the native "Update now" button
without any token in `wp-config.php`.
